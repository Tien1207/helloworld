# Hello World
## What you need to run the code:
A functional desktop computer

An environment that supports java

## Process:
To run the program, please download the code and open it in any development environment that supports java, such as IntelliJ IDEA.
Then simply compile the code, for example by clicking on the play button on the top right corner in IntelliJ.

You will see the result in the console, which pops up in the bottom of the window.
